# Support

Team Killing Bastards projects are independent community tools and are not officially supported by MissionChief or its operators.

## Before opening a support request

- Read the repository README and documentation.
- Confirm you are using the latest supported release.
- Check existing issues for the same behaviour.
- Reproduce the problem with unnecessary userscripts disabled where practical.

## Information required

Provide:

- Project name and exact version.
- MissionChief domain.
- Browser and version.
- Userscript manager and version.
- Steps to reproduce.
- Expected and actual behaviour.
- Relevant screenshots or console output with private information removed.
- Other userscripts running at the time.

## Where to ask

Use the issue forms in the affected repository:

- **Bug report** for reproducible incorrect behaviour.
- **Feature request** for a proposed improvement.
- **Security reporting** for sensitive vulnerabilities.

Do not include login credentials, session cookies, private API keys, access tokens or personal data.

## Support boundaries

Support is provided on a reasonable-effort basis. Maintainers may close requests that:

- Cannot be reproduced.
- Lack essential diagnostic information.
- Relate to unsupported or modified builds.
- Are caused by another script or browser extension.
- Request help bypassing game rules or platform restrictions.
