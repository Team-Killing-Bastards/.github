# Team Killing Bastards — Organisation Configuration

This repository controls the public Team Killing Bastards organisation profile and provides default community-health files for organisation repositories that do not define their own.

## Organisation profile

The public organisation homepage is generated from:

```text
profile/README.md
```

## Default community files

The repository provides organisation defaults for:

- Contribution guidance
- Security reporting
- Support requests
- Community conduct
- Governance and project authority
- Bug-report forms
- Feature-request forms
- Pull-request descriptions

Individual repositories may override these defaults with project-specific files.
